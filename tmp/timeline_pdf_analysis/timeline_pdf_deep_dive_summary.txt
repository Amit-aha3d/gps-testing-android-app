Recommended Actions
- Keep multi-endpoint rotation for 429 handling.
- Add a 429 cooldown per endpoint instead of trying the same endpoint again immediately on the next cycle.
- Separate GPS cache cadence from API query cadence so Overpass is not called every cache cycle.
- Add inferred speed defaults for roads where OSM has no maxspeed tag.
- Improve timeline export formatting so edge case and approach values do not wrap into ambiguous fragments.
